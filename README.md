# FGO展 グッズ購入リスト

スマートフォン向けのシンプルな購入リストWebアプリです。

## 使い方

1. `index.html`、`style.css`、`app.js`を同じフォルダに置きます。
2. `index.html`をブラウザで開きます。
3. 商品を選択し、数量とメモを入力します。
4. 画面下部の「購入リストをコピー」を押します。

## 主な機能

- 商品選択
- 数量入力と増減ボタン
- 商品ごとの自由記入メモ
- 合計金額の自動計算
- 商品検索・販売区分フィルター
- 選択済み商品の絞り込み
- 購入リストのクリップボードコピー
- 入力状態のブラウザ保存（localStorage）
- 公式グッズページへのリンク

## データについて

商品名と価格は、Fate/Grand Order展 -星見の回廊- 公式グッズページを基にしています。
販売状況や価格が変更される場合があるため、購入前に公式ページを確認してください。
